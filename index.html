Quebra-Cabeça de Fotos

Fase Um - Organize as peças numeradas de 0 a 8

Movimentos: 0

Como jogar:

1. Clique em uma peça adjacente ao espaço vazio para movê-la.

2. Organize as peças na ordem correta para formar a imagem completa.

3. A peça vazia (0) deve estar no canto inferior direito para vencer.
